# 👋 はじめまして

Web アプリ開発のフリーランスエンジニアです。
React / TypeScript を軸に、フロントエンドの設計〜実装をリードしつつ、バックエンド（Express）や DB 設計にも領域を広げています。

## 💼 About

- 経験: 約 2 年半（正社員 → フリーランス）
- 直近の役割: スタートアップの自社サービス 0→1 開発でフロントエンドリード（設計判断・レビュー・新人教育）
- 担当範囲: 要件定義 → アーキテクチャ設計 → 実装 → テスト → 運用まで一貫して対応
- バックエンド: Express による REST API の設計・実装・改修、DB 設計（ER 図・テーブル定義）、SQL による運用データ抽出
- AI 活用: Claude Code / Cursor / Codex 等を業務導入し、チーム運用ルールの設計・整備まで担当

## 🔨 実績

- 20 画面超の SPA を FSD アーキテクチャで設計・実装（リリース後 2 ヶ月で累計会員登録 1,800 名超）
- 4 社横断の大規模物流システムでフロントエンドリードとして期限内リリースを完遂
- 仕様未確定の 0→1 開発で、画面遷移図・状態遷移表・例外パターンを先行作成し、大規模な手戻りなくリリース
- GitHub Actions による CI/CD 構築、PR レビュー体制の整備

## 🚀 今後の方向性

フロントエンドの経験を活かしつつ、認証認可・例外設計・テスト戦略・ログ/監視といった基盤技術を実務で担当できるフルスタックエンジニアを目指しています。
