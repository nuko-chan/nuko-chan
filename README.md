# 👋 はじめまして

東京都在住のWebエンジニアです！
受託・自社開発企業の正社員 → 自社開発フリーランス。
直近 1 年はREST API 実装・改修、バリデーション仕様設計、DB 変更提案、SQL による運用データ提供を担当しつつ、フロントエンドのリード（設計判断・レビュー・新人教育）を兼務しています。


## 💼 私について

- 経験: 約2年半（正社員 → フリーランス）
- 直近の役割: スタートアップ自社サービス 0→1 開発で、フロントエンドリード + バックエンド（API 実装・仕様レビュー・DB 設計）を兼務
- バックエンド:
  - REST APIの設計・実装・改修（Express / Zod / PostgreSQL）
  - SQL による運用データ抽出・集計 → 非開発部門への定期レポート
- フロントエンド:
  - 20 画面超の SPA を FSD アーキテクチャで設計・実装（リリース後 2 ヶ月で累計会員登録 1,800 名超）
  - 4社横断の大規模物流システムでフロントエンドリードとして大規模な手戻りなく期限内リリースを完遂
- AI活用: Claude Code / Cursor / Codex 等を業務導入し、チーム運用ルールの設計・整備まで担当
- GitHub Actions による CI/CD 構築、PR レビュー体制の整備

## 🚀 キャリア方向性

フロントエンドの設計・実装経験を土台に、バックエンドの専門性確立を目指しています！
